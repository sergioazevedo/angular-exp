Quick Example ------------- ``` js var Showdown = require('showdown'); var converter = new Showdown.converter(); converter.makeHtml('#hello markdown!'); ``` What's it for? -------------- Developers can use Showdown to: * Add in-browser preview to existing Markdown apps Showdown's output is (almost always) identical to markdown.pl's, so the server can reproduce exactly the output that the user saw.
